# Gelar Cipta 2026 --- Design System

## 1. Design Identity

### Brand Name

Gelar Cipta 2026

### Design Concept

Luxury Dark Neo-Brutalist Culinary Exhibition Website

### Design Philosophy

Website harus memberikan pengalaman seperti mengunjungi pameran kuliner
premium.

Menggabungkan: - Dark neo-brutalist interface - Luxury editorial
magazine - Fine dining aesthetic - Contemporary art exhibition - Modern
academic showcase

Website bukan sekadar website acara, tetapi digital exhibition yang
menampilkan karya, kreativitas, dan inovasi mahasiswa.

# 4. Typography System

## Heading Font

Luxury editorial serif.

Recommended: - Playfair Display - Cormorant Garamond - Instrument Serif

Karakter: - Oversized - Bold - High contrast - Editorial feeling

## Body Font

Recommended: - Inter - Geist - Helvetica Neue

Karakter: - Clean - Minimal - Readable

    300-500ms

# 11. Image Direction

# 12. Responsive Design

------------------------------------------------------------------------

# 13. Technical UI Rules

Gunakan: - SVG icons - CSS animations - Semantic HTML - Accessible
contrast

Hindari: - Emoji icons - Excessive icon usage - Dashboard layout -
Bright neon colors

# 14. Final Design Goal

Website harus terasa seperti:

"A luxury culinary exhibition presented through a futuristic digital
gallery."

Kesan pengunjung: - Professional - Premium - Artistic - Memorable -
Prestigious

Gelar Cipta harus terlihat seperti showcase kuliner tingkat nasional,
bukan website acara kampus biasa.
