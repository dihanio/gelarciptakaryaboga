# Identitas Proyek
Gelar Cipta EMS dikhususkan untuk **S1 Pendidikan Tata Boga Universitas Negeri Surabaya (UNESA)**. 
- Seluruh default value, copywriting, branding teks, contoh input (jika dibutuhkan oleh logika sistem, bukan simulasi UI), header cetakan tiket, dan elemen identitas lainnya harus mencerminkan program studi dan universitas ini.
- Gunakan bahasa yang relevan dengan konteks akademik dan acara kuliner/tata boga.
